Create beautiful responsive image collages for all your posts and pages

== Description ==

Easy Image Collage is a user friendly plugin for adding image collages to any of your posts and pages.

Watch this video introduction to see it in action:

[youtube https://www.youtube.com/watch?v=uUEs98iGG_I]


== Installation ==

1. Upload the `easy-image-collage` directory (directory included) to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add image collages on the page or post screen
